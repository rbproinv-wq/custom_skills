## Step 6: Update `SKILL.md` — Add RICE Prioritization + Non-Goals + Pre-Pay Gate

**File:** `product-analyst-mvp/SKILL.md`

**Action:** Three additions.

### 6a: RICE Prioritization (new Step 5)
```markdown
### Step 5: RICE Prioritization

Score each proposed feature using RICE:
- **Reach:** How many users/month? (1-10)
- **Impact:** How much impact? (1-10 scale: 1=minimal, 10=game-changing)
- **Confidence:** How confident? (20%=wild guess, 50%=medium, 80%=high, 100%=data)
- **Effort:** Person-months? (XS=0.1, S=0.5, M=1, L=3, XL=8)

**Score = (Reach × Impact × Confidence) / Effort**

Include the score for each MVP feature. Features below threshold (e.g., <5) move to post-MVP or kill.
```

### 6b: Pre-Pay Gate (after RICE)
```markdown
**MVP Gate:** Before finalizing MVP scope, ask:
"Would 3 customers pay for this today?"
- If no: Re-scope MVP to something smaller or validate with landing page first.
- If yes: Proceed.
```

### 6c: Non-Goals (section 12 of PRD)
```markdown
**12. Não-Escopo (obrigatório)**
List at least 3 things explicitly NOT in scope:
- [ ] O que NÃO está incluso nesta versão
- [ ] O que será adiado para próxima iteração
- [ ] O que nunca será feito (para evitar scope creep)
```

**Verify:**
- ✅ RICE scoring present in workflow
- ✅ Pre-pay gate asked before finalizing MVP
- ✅ Non-Goals section in PRD template references
